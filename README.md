# نظام جرد حلويات الشيخ 🍰

نظام إدارة مالية متكامل واحترافي لمحل حلويات، يوفر واجهة سريعة تشبه الكاشير مع تقارير شاملة وتحكم كامل بالوصول.

## ✨ المميزات الرئيسية

### 🏪 واجهة الكاشير السريعة
- إدخال سريع للعمليات (إيراد/مصروف)
- حفظ فوري عند الضغط على Enter
- دعم طرق دفع متعددة (كاش، كليك، فيزا، بنك)
- تصميم احترافي يشبه أنظمة الكاشير العالمية

### 📊 التقارير الشاملة
- **جرد يومي:** عرض جميع العمليات مع ملخص يومي
- **جرد الكاش:** تتبع الأرصدة والحركات اليومية
- **تقارير شهرية:** إحصائيات شاملة لكل شهر
- **تقارير سنوية:** ملخص سنوي كامل
- **رسوم بيانية:** عرض بياني للمبيعات والمصاريف

### 🔒 نظام التحكم بالوصول
- قائمة بيضاء للمستخدمين المسموح لهم فقط
- ثلاث أدوار: مالك، مدير، موظف
- سجل محاولات الوصول الكامل
- تفعيل/تعطيل المستخدمين بسهولة

### 📱 التوافق والتصميم
- واجهة سريعة الاستجابة
- مناسبة للهاتف والكمبيوتر
- تصميم احترافي بألوان دافئة برتقالية

### 📤 الميزات الإضافية
- تصدير التقارير إلى PDF
- إرسال التقارير عبر WhatsApp
- إشعارات تلقائية
- لوحة تحكم شاملة

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- npm أو pnpm
- قاعدة بيانات MySQL/PostgreSQL

### التثبيت المحلي

```bash
# استنساخ المستودع
git clone https://github.com/your-username/sheikh-sweets-accounting.git
cd sheikh-sweets-accounting

# تثبيت الحزم
pnpm install

# إعداد قاعدة البيانات
pnpm db:push

# بدء خادم التطوير
pnpm dev
```

الموقع سيكون متاحاً على: `http://localhost:3001`

## 🛠️ التطوير والمساهمة

### الأدوات المستخدمة
- **Frontend:** React + TypeScript + TailwindCSS
- **Backend:** Node.js + tRPC
- **Database:** MySQL/PostgreSQL + Drizzle ORM
- **UI Components:** shadcn/ui
- **Charts:** Recharts
- **Auth:** Manus OAuth

### تشغيل الاختبارات

```bash
# فحص TypeScript
pnpm check

# اختبار الإنتاج
pnpm build
```
